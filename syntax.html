Syntax

This page defines the syntax available in the Caffeinated Markup specification.

Escaping

Escaping markup text in other languages can be a bit of a pain. Sometimes you seem to spend a lot of time fiddling around just to get a piece of markup-syntax to be printed as literal text without going though various acrobatic hoops.

To that end, escaping in Caffeinated Markup is designed to be as simple as possible so you can spend more time on your content and less time fighting your markup language!

Caffeinated Markup gives you two simple options for escaping:

  • Block escape
  • Single character escape

For those times when you just need to stop a sequence being interpreted as markup, use the single character escape: 

**This will be emphasised**, but \**this\** will just be surrounded by stars!

Of course, if there's a lot you need to escape it can be tedious let alone messy to have to escape every single sequence. For this eventuallity there's the block escape.

Consider the following: 

This \**has\** a lot of \{{literal markup\}} that we \[[don't want to be interpreted\//

That works, but this is much cleaner! 

%%This **has** a lot of {{literal markup}} that we [[don't want to be interpreted//%%

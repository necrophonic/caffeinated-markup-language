Caffeinated Markup

The Caffeinated Markup Language is an attempt to create a simple yet rich markup language for easy posting of awesome blog posts and articles that can then be published into a variety of formats.

Motivation

There are a lot of markup languages out there (HTML, Markdown, LaTEX etc) and they all have strengths and weaknesses. Caffeinated Markup doesn't attempt to replace any of these but provide another option building on the strengths found in other languages.

Example

# Caffeinated Markup is awesome
If you're looking for a **simple** yet **powerful** way to //markup// text
whilst leaving it relatively uncluttered with markup furniture, then
//Caffeinated Markup// is the choice for you!

""Caffeinated Markup makes me happy!|Happy Dude""

Output is dependant on the formatter implementation, but in HTML and plain text the output may look like:

HTML

<h1>Caffeinated Markup is awesome</h1>
<p>
If you're looking for a <strong>simple</strong> yet <strong>powerful</strong> way to
<em>markup</em> text whilst leaving it relatively uncluttered with markup furniture, then
<em>Caffeinated Markup</em> is the choice for you!
</p>
<blockquote>
Caffeinated Markup makes me happy!
<cite>Happy Dude""</cite>
</blockquote>

Plain Text

Caffeinated Markup is awesome
=============================

If you're looking for a simple yet powerful way to markup text
whilst leaving it relatively uncluttered with markup furniture, then
Caffeinated Markup is the choice for you!

Developers

This page is for those who are interested in developing parsers and formatters for Caffeinated Markup Language. Normal users should look at the syntax reference

Current version

The most up to date version of the spec is 1.0.0

All versions will be available in the git repo tagged in the master branch. The spec itself is available as a JSON file: spec.json

Versioning

Versions of the spec use the major.minor.patch convention.

  • major - Denotes major (breaking) changes. All versions with the same major number are backwards compatible. e.g. A 1.5.0 implementation must be able to correctly parse 1.1.0 compliant markup, but would not be expected to support 2.0.0
  • minor - Denotes minor (new feature) changes. e.g. A new markup class like "elephant formating" is added.
  • patch - Denotes a patch change to the spec. The actual spec won't change but may address typos etc. e.g. 1.3.1, 1.3.2, 1.3.3 are all considered to be the same for implementations.
